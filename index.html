🚀 Deploy (one-time, ~10 min)

  1. Deploy the Worker backend first (holds your API keys) — full steps in DEPLOY.md
  2. Paste your Worker URL into the WORKER_URL constant near the top of this file's <script>
  3. Go to github.com/new — create a public repo named job-tracker
  4. Upload this HTML file → rename it index.html → Commit
  5. Repo Settings → Pages → Source → Deploy from branch → main → / (root) → Save
  6. Your live URL:
    https://YOUR_USERNAME.github.io/job-tracker
